Source Code
-----------

The source is divided into 3 folders, they are:

* Empire - MicroSoft DX7 sdk stuff, I have made a few changes to some
 	   of this code. There's a lot of redundant code (and files)
 	   here.

* Rebels - My code.

* Yoda - Andre LaMothe's stuff. GPDUMB1 has been converted to DX7 and I
 	 ripped the sound code from GPDUMB2 (DumbSounds.cpp). Also
	 fixed a few errors, should work on all video cards now.
