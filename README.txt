			    ----------------
			    LAME CARZ 2 v1.1
			    ----------------

Lame Carz 2 is not any kind of "ware" (Freeware,Shareware,etc.), it's
only an exercise in DirectX programming. The source may be useful to a
beginner (I am new to DX and WIN32 programming myself).
 
LC2 has the following features:

* Streaming WAV loader (Ripped directly from the DX7 sdk).
* DirectMusic MIDI player (Ripped directly from the DX7 sdk).
* DirectPlay IPX conn. (This code was partially written by myself, 
  there is some good code in the sdk that I based it on).
* DirectSound.
* DirectDraw.
* The GPDUMB library with GPDUMB1 converted to DirectX7 and major
  errors fixed.

Instructions
------------
LC2 is a 2 player game that ONLY works on an IPX network.
Gain 8 flags (1 flag per lap) and complete a final lap to win a race!
Crashing into another car loses you a flag.

Keys:

(<-) 	Turn left
(->) 	Turn Right
(Space) Accelerate
(Ctrl) 	Brake 
(S)	Start in-game music
(Esc) 	Quit

Known Bugs
----------
* DirectMusic/Sound memory leak? hmmm - After executing several times,
  the music and sound starts to crackle - WORSE if you use the in-game
  music.
* Mouse pointer shows up uninvited on some video cards.
* There may be other mem. leaks!
